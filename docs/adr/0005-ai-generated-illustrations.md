# ADR-0005: Генеративные иллюстрации с ИИ-ревью на артефакты

**Статус:** принято · Этап 4

## Контекст
Лендинг нуждался в профессиональном виде; первый вариант с «мультяшными» SVG отвергнут владельцем. Запрошены генеративные растровые иллюстрации, причём с ревью, чтобы исключить ИИ-артефакты (лишние пальцы, кривой текст). Прямой keyless-генератор (Pollinations) оказался недоступен (paywall/лимит), публичный OpenAI image-API недоступен на ChatGPT-авторизации.

## Решение
**`codex image_gen` (GPT Image)** — встроенная генерация Codex, работает на ChatGPT-авторизации. Анти-артефактная стратегия: **только предметные сцены** (стол, шары, сукно, свет) **без людей, рук, лиц и текста** + усиленные негативные промпты (`perfectly round undistorted balls, no warped/merged geometry`). **Ревью-петля:** каждое изображение визуально отсматривается на артефакты; неудачное — перегенерировать с уточнением. Итог оптимизирован PNG→JPG (≈5 МБ → ~0.35 МБ).

## Последствия
- (+) Реальные фотореалистичные ассеты без артефактов; самодостаточный быстрый лендинг.
- (+) Воспроизводимый процесс «сгенерировать → отсмотреть → принять/перегенерировать».
- (−) Зависимость от доступности `image_gen` через codex; стиль ограничен «безопасными» предметными сценами.
- ТЗ на дизайн выработано совместно (Claude+Codex), вёрстку собрал оркестратор вручную.
