# Хронология разработки

Документ фиксирует эволюцию проекта и **способ** его создания (через ИИ-агентов). Технические детали — в `spec/` и `adr/`.

## Этап 0 — Хобби-прототип
Зеро-зависимостей Node.js (`http`+`fs`, ~177 строк), данные в JSON-файлах (`data/players.json`, `games.json`, `series.json`), без авторизации, один экран, учёт партии с одного устройства. Уже содержал доменную логику: шары/очки, дуплеты, штаны, штрафы, очерёдность ходов.

## Этап 1 — Публикация
Развёрнут на сервере: Docker-контейнер `node:24-alpine`, nginx как reverse-proxy, HTTPS (Let's Encrypt), поддомен `tashkent.spellful.site`. Данные приложения защищены от затирания при `git pull` (`skip-worktree`). См. ADR-0007.

## Этап 2 — Превращение в платформу (по запросу владельца)
Большой апгрейд, выполнен поэтапно с ревью постановки:
- **БД:** переезд JSON → PostgreSQL, миграция данных без потерь (3 игрока, 2 игры, 60 событий — сверено). ADR-0001.
- **Авторизация:** Keycloak (OIDC), регистрация, сессии в Postgres. ADR-0002.
- **Real-time:** WebSocket — несколько телефонов в одной партии. ADR-0003.
- **Доменная модель:** игрок ↔ аккаунт, игроки-гости, автозаход в активную игру по составу, объединение профилей, видимость игр по составу. ADR-0006.

## Этап 3 — AI-native: самодорабатываемый продукт
В приложение встроен **«Чат с разработчиком»** (только для владельца): сообщение → очередь `admin_tasks` → хост-воркрер (systemd) запускает **Codex CLI** над репозиторием → правки → commit/push → передеплой. Владелец начал **дорабатывать продукт сам, на естественном языке**. Через этот канал сделаны: перевод интерфейса на колонки игроков, очки `+/−`, взаимозачёт, новые режимы истории, круглые кнопки одной буквой, правило золотого шара (15 шаров суммарно, по 5), загрузка и **анализ фотографий** в чате (codex читает изображения с диска). ADR-0004.

## Этап 4 — Зрелость дизайна (ИИ-ассистированно)
- Несколько итераций лендинга; первый «мультяшный» вариант отвергнут владельцем как непрофессиональный.
- Совместная (Claude+Codex) выработка дизайн-ТЗ; **иллюстрации сгенерированы нейросетью** (`codex image_gen` / GPT Image) с **ревью на ИИ-артефакты** (предметные сцены без людей/рук/текста → исключение «лишних пальцев»). ADR-0005.
- Полный рестайл всех поверхностей (приложение + тема Keycloak) под единую премиальную дизайн-систему.
- Донаты на всех страницах (кроме авторизации) + CTA-благодарность на экране итогов партии/серии.

## Сквозная линия
От «скрипта на коленке» к **AI-native платформе**, где значимая часть поздней разработки выполнена **самим владельцем через ИИ-агентов**. История в `git log` (коммиты с префиксом `admin-chat:`) — это фактические промпты владельца; их эволюция — материал для оценки обучения (см. `evaluation.md`).
