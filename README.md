# Ташкент — учёт игр

Локальное веб-приложение для учёта забитых шаров в «Ташкенте». Хранит игроков и игры в JSON-файлах в `data/`.

## Запуск

```bash
node server.js
```

Открыть: <http://localhost:7777>

Порт меняется через `PORT=8888 node server.js`.

## Правила (используются в учёте)

- Победа: первый, кто наберёт **N шаров** (по умолчанию 6, настраивается при старте).
- Действия:
  - **Обычный шар** — +1 шар, +1 очко, ход остаётся.
  - **Дуплет** — +1 шар, +2 очка, ход остаётся.
  - **Штаны** — +2 шара, +3 очка, ход остаётся.
  - **Промах** — счёт не меняется, ход переходит.
  - **Штраф** — −1 шар у текущего игрока, ход переходит.
- Очерёдность ходов — по порядку, заданному при старте.

## Данные

- `data/players.json` — справочник игроков.
- `data/games.json` — все игры (события, итоговые счёта, победители).

Любую игру можно открыть из истории и удалить.
